What's in the box

A complete language, not a toy.

⚙️

Bytecode VM

Source compiles to opcodes and runs on a stack VM. Tail-call optimized. Also compiles to WebAssembly.

🧩

Full language

Functions, closures, classes + inheritance, generators, pattern matching, decorators, comprehensions, typed exceptions.

🔌

C FFI

Call any native library — and pass a LIPI function back as a C callback. Cross-platform (Windows/Linux/macOS).

🌐

Networking

HTTPS/TLS, JSON, an HTTP server, sockets, threads — enough to talk to modern APIs and LLMs.

📐

Mission-critical stdlib

Units & dimensional analysis, linear algebra, quaternions, PID + Kalman control, geodesy, ECC, intervals.

🛠️

Tooling

Browser IDE, formatter, linter, doc generator, profiler, step debugger, LSP, package manager.

Install

Pick your platform.

git clone https://github.com/Nemi-swami/LIPI-Hindi-Programing-LAng
cd LIPI-Hindi-Programing-LAng
cargo build --release
# binary: target/release/lipi

Pure Rust, no external crates. Builds on Windows, Linux, and macOS. Requires Rust (rustup.rs).

curl -fsSL https://raw.githubusercontent.com/Nemi-swami/LIPI-Hindi-Programing-LAng/main/install.sh | sh

Builds from source with Cargo and installs lipi to ~/.local/bin. Requires Rust and git.

# PowerShell:
irm https://raw.githubusercontent.com/Nemi-swami/LIPI-Hindi-Programing-LAng/main/install.ps1 | iex

Or download LIPI-Setup.exe from the Releases page and run it.
