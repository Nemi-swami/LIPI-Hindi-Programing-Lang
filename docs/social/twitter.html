🧵 LIPI — a Devanagari-syntax language, in pure Rust

1/6

LIPI lets you program with Hindi/Sanskrit keywords. It's not a toy — source compiles to a bytecode VM (LVM) and also runs in the browser via WebAssembly. Hello world:

बताओ "नमस्ते दुनिया"
→ नमस्ते दुनिया
2/6

Functions, classes, closures, generators, pattern matching, decorators — the full toolkit. Higher-order functions too:

बताओ छानो([1,2,3,4,5,6], लाम्डा(x): x % 2 बराबर 0)
→ [2, 4, 6]
3/6

Exact math when you need it — arbitrary-precision integers, no float error:

आयात भारत.बड़ी
बताओ महा_भाज्य("30")   # 30!
→ 265252859812191058636308480000000
4/6

No ceiling: a C FFI. Call any native library — and pass a LIPI function as a C callback. C's qsort, sorted by a LIPI comparator:

# qsort([5,2,8,1,9], LIPI comparator)
→ [1, 2, 5, 8, 9]
5/6

Real HTTPS — so it talks to modern APIs and LLMs:

आयात भारत.सुरक्षित
बताओ https_पाओ("https://api.github.com/zen",
              {"User-Agent":"LIPI"})["स्थिति"]
→ 200
6/6

Also in the box: JSON, CSV, regex, SQL, ZIP, sockets, threads, an HTTP server, units + linear algebra + Kalman control, a browser IDE, formatter, linter, profiler, debugger, LSP, package manager. Open source, 60+ passing examples. Contributors welcome.
